Latex Formula Editor
====================

About Latex Formula Editor
--------------------------

If you wish to inset a mathematical expression annotation you can use this editor to create a LaTeX rendered image. This image can then be inserted into an image annotation. This tool is a very nice stand-alone LaTeX editor that allows the user to copy a mathematical image to the clipboard or save it as an image file.  The version we incuded with this help system is a (very slight) alternation of the Latex Formula Editor by moskensoap on GitHub.com  Copyright (c) 2025, downloaded (6/2026).
